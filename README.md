# AutoKeyBot - Бот для автоматической продажи ключей

# Что делает бот:
* Принимает платежи через ЮKassa (бывшая Яндекс.Касса)

* После оплаты мгновенно выдает уникальный ключ

* Проверяет статус платежа

* Ведет статистику продаж

* Работает 24/7 автоматически

# 📋 Инструкция по запуску и монетизации

## Шаг 1: Регистрация и настройка

### 1. Создайте бота в Telegram:

  - Напишите @BotFather

  - Создайте бота командой /newbot

  - Получите токен и вставьте в .env файл в корне проекта:

    ```
    # Токен бота от @BotFather
    TG_BOT_TOKEN = "ВАШ_ТОКЕН_БОТА"
    # ID администратора (можно узнать у @userinfobot)
    TG_ADMIN_ID = 123456789  # Замените на ваш Telegram ID
    ```

### 2. Зарегистрируйтесь в ЮKassa:

  - Перейдите на yookassa.ru

  - Создайте магазин

  - Получите Shop ID и Secret Key

  - Вставьте их в .env файл в корне проекта:
    ```
    # Данные от ЮKassa (https://yookassa.ru)
    YK_SHOP_ID = "ВАШ_SHOP_ID"
    SECRET_KEY = "ВАШ_SECRET_KEY"
    ```

## Шаг 2: Подготовка товаров

   ### 1. Отредактируйте ```products.json``` под свои товары

   ### 2. Добавьте ключи в базу:
   
   ```
   # Через админ-команду в боте
   /add_keys product_1 KEY1,KEY2,KEY3
   ```

## Шаг 3: Запуск
```
# Установка зависимостей
pip install -r requirements.txt

# Запуск бота
python bot.py
```
